One improvement before we start

I'd like to make one architectural improvement to avoid future problems.

Instead of every engine reading from the previous engine's CSV with slightly different columns, we'll define a single standard schema.

For example:

Category	Standard Columns
Identity	Rank, Symbol
Market	CMP, Price Chg %, OI Chg %, Volume Chg (%)
Analysis	NTIS Score, Signal, Pattern, Pattern Reason
Prediction	BUY Probability %, SELL Probability %, Confidence, Trade Bias
Validation	Entry Price, Exit Price, Return %, Outcome
Learning	Pattern Accuracy %, Historical Win %, Model Score

Each engine can add columns, but it won't rename or remove existing ones. This consistency will prevent issues like the CMP vs Close mismatch.
